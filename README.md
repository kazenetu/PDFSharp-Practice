# PDFSharp-Practice
PDFSharpを利用した試作

## 実行方法
* コンソールアプリ実行
  * .NET6  
    ```dotnet run --project Console/dotnet6/console.csproj```
  * .NET7  
    ```dotnet run --project Console/dotnet7/console.csproj```
  * .NET8  
    ```dotnet run --project Console/dotnet8/console.csproj```
